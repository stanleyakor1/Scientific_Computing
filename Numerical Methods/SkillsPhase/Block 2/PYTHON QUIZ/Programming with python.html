Scipy (scientific python) is the python module for computational sciences. To see the list of the function available in scipy, first import scipy. The scipy and the numpy (numerical python) modules are very similar although scipy is more powerful. Solve ODE in python¶

An example of using ODEINT is with the following differential equation with parameter k=0.3, the initial condition y0=5 and the following differential equation.

\begin{equation} \frac{dy}{dt} = -ky \end{equation}

import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt

# function that returns dy/dt
def model(y,t):
    k = 0.3
    dydt = -k * y
    return dydt

# initial condition
ya = 5

# time points
a = 0
b = 20
N = 70
t = np.linspace(a,b,N)

# solve ODE
y = odeint(model,ya,t)

# plot results
plt.plot(t,y)
plt.xlabel('time')
plt.ylabel('y(t)')
plt.show()

Three-Dimensional Plotting in Matplotlib¶

Matplotlib was initially designed with only two-dimensional plotting in mind. Around the time of the 1.0 release, some three-dimensional plotting utilities were built on top of Matplotlib's two-dimensional display, and the result is a convenient (if somewhat limited) set of tools for three-dimensional data visualization. three-dimensional plots are enabled by importing the $mplot3d$ toolkit, included with the main Matplotlib installation:

In [28]:
